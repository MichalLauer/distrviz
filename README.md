# distrviz

<!-- badges: start -->
<!-- badges: end -->

The goal of distrviz is to vizualize (hopefully) every distribution that
is available in the [`{distr6}`](https://github.com/xoopR/distr6)
package.

![](assets/readme_front.png "Normal distribution")

The main features are:

- Support both **discrete** and **continuous** random variables (RV)
- Interactive chart of **probability distribution function (PDF)**
- Densities for continuous RV, histograms for discrete RV
- Interactive of **cummulative distribution function (CDF)**
- **Dynamic recalculation** of parameters
- For example, changing the *variance* of a normal distributions
  updateâ€™s the standard deviation and precision.
- Basich characteristics

The app can be seen live
[here](https://distrviz-ctt2nrcxzq-ey.a.run.app/). Note that it may take
longer time to load due to budgeting.
